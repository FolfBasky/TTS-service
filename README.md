# TTS Service — озвучка учебных тем

Загружает `.docx` с нумерованными темами, разбивает на отдельные темы, заменяет греческие буквы и спецсимволы на произносимые аналоги, озвучивает через TTS и сохраняет в MP3.

## Пример входа

Файл `.docx`:
```
17. Интегральная теорема Муавра-Лапласа, ЦПТ
Формула: P(a ≤ k ≤ b) ≈ Φ((b - np)/√(npq)) - Φ((a - np)/√(npq))
...
```

Выход: `17_Интегральная_теорема_Муавра_Лапласа_ЦПТ.mp3`

## Установка

```powershell
pip install -r requirements.txt
```

## Запуск (веб-интерфейс)

```powershell
python web.py
```

Открыть `http://127.0.0.1:8000` — загрузить `.docx`, дождаться обработки, скачать MP3.

## Запуск (консоль)

```powershell
python main.py путь_к_файлу.docx [папка_вывода]
```

## Замена символов

| Символ | Произношение |
|--------|-------------|
| θ, Θ | тэта |
| μ | мю |
| σ, Σ | сигма |
| λ | лямбда |
| π | пи |
| φ, Φ | фи |
| √ | корень квадратный |
| ≤ | меньше или равно |
| → | стремится к |
| M[X] | мат. ожидание |
| D[X] | дисперсия |
| P(A) | вероятность |

## TTS

- **Основной**: `edge-tts` (голос SvetlanaNeural, требуется интернет)
- **Резерв**: `pyttsx3` (офлайн, системный SAPI5-голос)

## Структура

```
├── parser.py       # Парсинг .docx, разбивка на темы
├── formatter.py    # Замена символов + имена файлов
├── tts_engine.py   # TTS (edge-tts → pyttsx3)
├── web.py          # FastAPI сервер
├── main.py         # Консольная версия
├── templates/
│   └── index.html  # Веб-интерфейс
└── requirements.txt
```
